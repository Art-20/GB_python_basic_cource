task_1_1
Пользователь вводит продолжительность периода в секундах. Программа разбивает на сутки, часы, минуты, секунды.
При вводе отрицательного значения - программа запрашивает повторный ввод значения.

task_1_2
Создать список, состоящий из кубов нечётных чисел от 1 до 1000.
a. Вычисление суммы тех чисел из  списка, сумма цифр которых делится нацело на 7
   Выводится исходный список для расчета и результат.
b. Создается новый список, состоящий из кубов нечётных чисел от 1 до 1000 и к каждому числу из списка прибавлено 17.
   далее алгоритм как в пункте a. В алгоритме более лаконичная запись, применяется +=
   Выводится исходный список для расчета и результат.
с. Создается список как в пункте a, далее повторяется расчет как в пункте a. После без создания нового списка происходит выполнение алгоритма с добавлением к каждому числу из списка 17.
   Выводится исходный список для расчета и два результата.

task_1_3
Реализовано склонение слова «процент» во фразе «N процентов».
Результат выводится на экран отдельной строкой для каждого из чисел в интервале от 1 до 100
Алгоритм подходит для любых интервалов целых чисел. Склонение реализовано изменением окончаний,
можно изменить 'процент' на другие единицы 'метр', 'литр', 'грамм' и т.д. алгоритм будет выполнятся.
